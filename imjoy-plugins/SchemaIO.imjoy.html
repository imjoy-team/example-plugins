# Introduction Input and output schema use this plugin. ### API #### clear_view (id) Clear up the input view **id** `int` view id #### vega_view_insert (args) Insert vega data to the chart ##### args `object` - id: `int` vega view index - data: `array` vega data value - dataName: `string` vega data name e.g. ~~~ { id: 0, data: [{ "category": "C", "position": 0, "value": 0.6 }], dataName: 'table' } ~~~ #### vega_schema_output (args) Output vega schema data, page display chart ##### args `object` - schema: `object` vega schema - message: `string` chart description - Id: `int` vega view index - title: `string` view title - show: `boolean` show status - buttons: `array` button list e.g. ~~~ { schema: vega schema, message: '', id: 1, title: 'vega output', show: true, buttons: [] } ~~~ #### form_schema_input (args) / run (args) Add input pannel / init SchemaIO window ##### args `object` - schema: `object` form-schema - model: `object` form data - callback: `object` submit function - title: `string` view title - show: `boolean` show status - formOptions: `object` form options - id: `int` view id e.g. ~~~ { schema: schema, model: { DNA: 'GGAGGCG...', modelType: 'all species' }, callback: plugin.show_schema_data1, title: 'input vue schema', show: true, formOptions: { validateAfterLoad: true, validateAfterChanged: true }, id: 1 } ~~~ #### Input Form Schema **GitHub:** https://github.com/vue-generators/vue-form-generator **Example:** https://jsfiddle.net/zoul0813/d8excp36 #### Output Vega Schema **Github:** **Example:** { "name": "SchemaIO", "type": "window", "tags": [], "ui": "", "version": "0.2.6", "api_version": "0.1.3", "description": "[TODO: describe this plugin with one sentence.]", "icon": "extension", "inputs": null, "outputs": null, "env": "", "requirements": [ "https://cdnjs.cloudflare.com/ajax/libs/vue/2.5.22/vue.min.js", "https://unpkg.com/vue-form-generator@2.3.4/dist/vfg.js", "https://unpkg.com/vue-form-generator@2.3.4/dist/vfg.css", "https://cdn.jsdelivr.net/npm/vega@5.3.4/build/vega.js", "https://cdn.jsdelivr.net/npm/vega-lite@3.1.0", "https://cdn.jsdelivr.net/npm/vega-embed@4.0.0" ], "dependencies": [], "defaults": {"w": 20, "h": 10}, "runnable": false }
