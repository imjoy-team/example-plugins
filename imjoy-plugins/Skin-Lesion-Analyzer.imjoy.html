This plugin allows to classify 7 types of skin lesions as described in the publication [The HAM10000 Dataset: A Large Collection of Multi-Source Dermatoscopic Images of Common Pigmented Skin Lesions](https://arxiv.org/abs/1803.10417) Images are analyze with a pretrained model built with Tensorflow.js. When loading the plugin, a default image is provided. You can also use your own images: on **computer** you can upload an image, on a **smartphone** you can snap an image. # Source code This plugin is ported from: [Skin Lesion Analyzer Web App built using Tensorflow.js](https://github.com/vbookshelf/Skin-Lesion-Analyzer).
